Performance and optimization

Nov 11th, 2019

Performance and optimization of your website are important because it can determine the number of people visiting your website, the traffic flow on your website and the usability experience of your website. Today I will discuss the tools I used when validating and optimizing my website portfolio.

Validating your code

When you create a website, you must always optimise the website by validating the code on the website this helps prevent any unwanted error. A website I like to use is The W3C Markup Validation Servicenot only does it identify any problems in the code, but it also makes it easier to maintain the website. You can validate the website either by entering the URI of the website, you can upload the file from your computer, or you can use the direct input option to directly enter the code on the validation service website. The validation service does not have to be only used for validating your website it can also be used to identify any errors in the code that are preventing it from working.

Website Optimization

Website optimization is the process of improving the SEO score by increasing the rank of the website in the search results. Nevertheless, websites with high ranking have a higher chance of being placed further up the search results (Admin, 2019). The website I used to optimize my website is Seoability.This website is simple to use you just enter the website domain it will identify SEO issues and then highlights the most important issues that are affecting your website. Seoability scores your website based on meta-information, page quality,page structure, link structure, server and external factors.Each of these sections are scored separately and you get an overall score for the webpage.The score helps indicate the quality of the webpage. Apart from the optimization of the website I compressed the images using Tiny jpg. This website helps reduce the file size of the images and it's very simple to use just drag and drop your image and it will compress the image size for you.

Search engine optimization

Search engine optimization (SEO) is very important to a developer like me because using SEO correctly can increase the traffic on the website. SEO is very important because 93% of online experiences begin in search engines and around 67,000 searches occur every second on Google and 46% of those searches are local (Nikolovska, 2019). SEO helps with the ranking of a website on search results. Ranking of a website depends on many parameters such as searches that are relevant to a particular search are displayed first. Using keyword optimization can increase the traffic on your website because it makes it easier for the search engine to know what the topic of your website/page in about (Admin, 2019. Good SEO can help with providing good user experience and increased usability of a website.
