My work

This project was part of a university group project and the aim was to create a website on activities for new students to do in Manchester. My group choose exploring street art as the activity and our website is about street art in Manchester. Since this was a group project, I created the ‘map’ page and it was created using HTML and CSS.

In this project I had to create an app on a mobile device so users could vote for their favourite pop star. This app was created with the help of code studio using PHP. Once the user has selected their favourite pop star, they can press the update chart button and the chart will add the users vote and display a pie chart with the results.

This is an order form which allows users to order shirts and ties. This form was created using JavaScript. Users have the opportunity to select their customer status and get discount e.g. registered customers save 10% but trade customers save 20%. There is an opportunity to select delivery options e.g. delivery within 7 days is £2 but next day delivery is £5.

My portfolio is the most recent project I have worked on. This portfolio gives me an oppertunity to show my coding skills and for people to know a little bit about me and my work.
