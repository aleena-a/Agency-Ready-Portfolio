Building my portfolio

Nov 2nd, 2019

CSS grid is more flexible to use especially when information needs to be divided up into sections, unlike flexbox it's limited to where it can move.

My research

What is a professional website portfolio and what does it include? To answer this question, I researched the internet and according to the University of California Berkeley “A professional portfolio, in either electronic or hard-copy format, showcases your skills, experiences and accomplishments to a potential employer. It can include pieces from your professional work, education and training, volunteer experience and personal interests” The University of California Berkeley, no date). So, this answered what I needed to include in my website portfolio. However, this is not enough to create a professional portfolio, therefore, I carried out further research to understand the different portfolios available online. The purpose of my portfolio is to show my work and my skills by building my website portfolio. A similarity I found in many of the portfolios I looked at was, the portfolio displayed the individual's personality, for example, Robby Leonardi has laid his portfolio out like a videogame that users can play but at the same time the portfolio displays information. This suggests he enjoys videogames. Another example is Sean Halpin's portfolio which displays his creative side because he uses animation and lots of colour on his website, but the website also displays his individuals’ skills. Another similarity is many websites include the ‘contact’ page this page allows future employees or users of the website to get in contact with the portfolio creator. I ensured on my website portfolio I included a ‘contact’ page. First, I created it as an HTML page however the information was not being stored or sent anywhere. I wanted to make use of the information gathered in the contact form, so I recreated the page using PHP. This has now allowed me to link my email address to the contact form, and now every time a user fills in the contact form I receive an email on my email address to tell me a user has filled in the form and I can see their message and reply. Thus, I can communicate with users who will view my portfolio.

CSS grid vs Flexbox

Whilst carrying out my research I realised many of the portfolios use CSS grid for layout of their webpages. Initially, when creating my CV page, I wanted to use flexbox however I ended up using CSS grid to create my CV. The difference between grid layout and flexbox is CSS grid is for 2-dimensional layouts and when you want to design the webpage using a mixture of rows and columns whilst flexbox is for 1-dimensional layout and anything that needs to be aligned responsively. (Andrew, no date).The CV is split into sections and CSS grid makes it much easier to lay the sections out. CSS grid layout is useful because it allows you to split the page into sections unlike flexbox and each box can be a different size. You can add rows and columns onto a grid so you can design and build your website more easily because you can separate the order of elements. You do not have to compromise on the structure of your documents, and you don’t have to worry about text overlapping when you change the text font size because the columns and rows can be modified. Nevertheless, CSS grid is supported by most browsers around the world.

To create the CV, I used 7 CSS grid boxes and it allowed me to split the information into 2 sections one side has quick points I want the user to read such as skills, languages that I speak and my contact details. Whilst the other side contains more important detail such as my work experience and my education. I have used fraction (Fr) units in my code when deciding the size of the columns because Fr units calculate the size of the column/row based on the amount of available space. It will do this by deducting the grid-gap from the total width and then calculate the size of the column/row. On the other hand, percentages can be used to determine the size of the column/row but unlike Fr units it doesn’t take into account the grid gaps Percentages can also be used however the width of the column is calculated based in the width of the container and it doesn’t take into account the grid gaps so the content will be pushed outside the grid gaps(Rahangdale, 2019).
